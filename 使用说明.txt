家庭任务管理系统 v2.0.1 使用说明
========================================

🎉 欢迎使用家庭任务管理系统！

📋 系统概述
--------
这是一个专为家庭设计的简洁任务管理系统，支持多成员协作、任务分配、
日历规划、重复任务设置、完成统计等功能。采用冷色调可爱风格设计，
界面简洁清爽，支持多设备实时同步。

⚡ 快速安装
--------
1. 确保已安装 Python 3.7 或更高版本
2. 双击运行 install.bat 进行自动安装
3. 安装完成后，桌面会出现"家庭任务管理系统"快捷方式

🚀 快速启动
--------
方式一：双击桌面快捷方式
方式二：双击 start_server.bat
方式三：命令行执行 python server.py

启动后浏览器会自动打开系统页面
本设备访问：http://localhost:8080
其他设备访问：http://[显示的IP地址]:8080

📱 主要功能
--------
✅ 家庭成员管理（增删改查）
✅ 任务管理（待办/已完成/已放弃三种状态）
✅ 优先级设置（高/中/低三级）
✅ 日历任务规划（周视图/月视图）
✅ 任务重复功能（每天/每周/每月等）
✅ 任务完成统计（排行榜显示）
✅ 多设备实时同步（3秒自动同步）
✅ 数据持久化保存（刷新不丢失）

🔧 基本操作
--------
• 添加成员：点击"添加成员"按钮，输入姓名
• 创建任务：点击"添加任务"按钮，填写信息并分配成员
• 完成任务：点击任务右侧的绿色"✓"按钮
• 放弃任务：点击任务右侧的红色"✕"按钮
• 重新激活：在已完成/已放弃任务中点击"↻"按钮
• 编辑任务：点击任务右侧的编辑按钮（笔图标）
• 删除任务：点击任务右侧的删除按钮（垃圾桶图标）

📅 日历功能
--------
• 点击顶部"日历计划"按钮进入日历界面
• 选择成员后可查看其任务安排
• 在日期格子中点击"+ 添加"可新建任务
• 勾选"开启重复功能"可设置任务重复规则
• 支持每天、每周、每两周、每月等重复模式
• 点击"回到当前月/周"按钮可快速回到当前时间

📊 统计功能
--------
• 点击顶部"任务统计"按钮查看完成情况
• 选择成员和时间段（周/月）查看统计
• 显示总完成次数和不同任务数
• 按完成次数从高到低排列任务

🌐 多设备同步
--------
• 确保所有设备连接到同一WiFi网络
• 主设备启动服务器后记录显示的IP地址
• 其他设备浏览器访问该IP地址
• 任何设备的修改会在3秒内同步到所有设备

💾 数据说明
--------
• 数据存储在 shared_data.json 文件中
• 本安装包已包含当前的用户数据
• 建议定期备份 shared_data.json 文件
• 系统支持本地和服务器双重数据保存

⚠️ 注意事项
--------
• 首次使用前请确保已安装Python环境
• Windows防火墙可能会询问是否允许Python程序通过，请选择"允许"
• 多设备使用时请确保在同一WiFi网络下
• 数据修改后约3秒会自动同步到其他设备

🔄 卸载方法
--------
如需卸载系统：
1. 运行 uninstall.bat
2. 确认删除操作
3. 系统将完全移除所有文件和快捷方式
